## PR 규칙

- 브랜치명: `챕터번호/닉네임`(e.g. 02/sun)
- 파일명: `my-sql/chapter{N}/` 경로의 `N_제목_이름` (e.g. my-sql/chapter02/02\_설치와설정_전선규.md)

- PR명: 제목 앞에 `[N장]` 포함, 이외 자유(e.g. [2장] 설치와 설정)
